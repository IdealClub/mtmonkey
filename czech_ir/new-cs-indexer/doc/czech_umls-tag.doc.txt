Czech UMLS Gazetteer Pipeline
==============================================

Requirements:
* Unix/Linux
* GATE

This pipeline performs annotation of UMLS terms in Czech documents.
It consists of the following Processing Resources (PR):
1) Reset:Czech_UMLS
2) Mesh lemma Gazetteer
3) Mesh form Gazetteer
4) Transfer Morpho/Sentence to mimir

Location:
* czech_umls-tag/pipeline.gapp

Reset:Czech_UMLS
----------------
    Requirements:
    * ANNIE Document Reset PR

    Removes the annotation sets: mimir, plain.

Mesh lemma Gazetteer
----------------------
    Requirements:
    * Flexible Gazetteer
    * ANNIE Gazetteer

    Annotates the lemmatized text (in their base forms) with UMLS (Mesh) terms.

    Input:
    * annotation Morpho/Token
        - Features:
            * clean_lemma

    Output:
    * annotation mimir/Lookup
        - Features:
            * majorType = "czmesh_lemmas"
            * meshID
            * czTerm
                - Czech UMLS term
            * enTerm
                - corresponding English UMLS term

Mesh form Gazetteer
----------------------
    Requirements:
    * ANNIE Gazetteer

    Annotates the original text with UMLS (Mesh) terms.

    Input:
    * annotation Morpho/Token

    Output:
    * annotation mimir/Lookup
        - Features:
            * majorType = "czmesh"
            * meshID
            * enTerm
                - corresponding English UMLS term

Transfer Morpho/Sentence to mimir
------------------------------------
    Requirements:
    * Annotation Set Transfer

    Transfers the annotation of sentences to the "mimir" annotation set.

    Input:
    * annotation Morpho/Sentence

    Output:
    * annotation mimir/Sentence
