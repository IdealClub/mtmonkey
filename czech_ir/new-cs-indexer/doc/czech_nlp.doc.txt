Czech NLP Pipeline
==============================================

Requirements:
* Unix/Linux
* GATE

This is a Czech NLP pipeline performing sentence splitting, tokenization and POS tagging.
It consists of the following Processing Resources (PR):
1) Reset:Czech_NLP
2) cz:TreexSegmenter
3) cz:MorphoDiTa
4) Clean lemmas

Reset:Czech_NLP
----------------
    Requirements:
    * ANNIE Document Reset PR

    Removes all annotations of the types: Sentence, SpaceToken, Token and Split.

cz:TreexSegmenter
-----------------
    Requirements:
    * perl 5.12
    * CPAN modules:
        * List::Util

    Location:
    * plugins/TreexSegmenter

    Install:
    1) Copy it to the $resourceshome$ dir
    2) run:
        cd TreexSegmenter
        ant

    This PR runs a sentence splitting (segmentation) algorithm on a given GATE document/corpus.
    The algorithm is implemented in Perl and called as a subprocess from the Java wrapper. The subprocess
    is executed only once in the init stage. To indicate the end of a document, a special character sequence
    "<__DOCUMENT_END__>\n" is sent to the subprocess' standard input. One should make sure this character
    sequence does not appear in any document to be processed.

    Input:
    * content
    * no annotation needed

    Output:
    * annotation Morpho/Sentence
        - labels a sentence
    * annotation Morpho/SpaceToken
        - labels whitespaces outside a sentence
    * annotation Morpho/Split
        - labels newlines

cz:MorphoDiTa
------------------
    Requirements:
    * gcc 4.7.3

    Location:
    * plugins/MorphoDiTaPlugin

    Install:
    1) Copy it to the $resourceshome$ dir
    2) run:
        cd MorphoDiTaPlugin
        ant
        # ant -Dbits=32   # compile a MorphoDiTa library for 32bit machines even if run on a 64bit machine
        # ant -Dbits=64   # compile a MorphoDiTa library for 64bit machines even if run on a 32bit machine
    3) before running GATE (replace $resourceshome$ with its concrete value):
        export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:$resourceshome$/MorphoDiTaPlugin/lib

    This PR is a GATE wrapper of MorphoDiTa: Morphologic Dictionary and Tagger (http://ufal.mff.cuni.cz/morphodita).
    It runs a Czech tokenizer and POS tagger on a document. cz:MorphoDiTa can possibly run also sentence splitting
    over the document, however the segmentation returned by cz:TreexSegmenter is of better quality. It is controlled
    by a run-time parameter "splitSentences", which is "false" if annotation of sentences is provided (default).

    Input:
    * content
    * annotation Morpho/Sentence
        - if splitSentences=false

    Output:
    * annotation Morpho/Sentence
        - labels a sentence
        - if splitSentences=false
    * annotation Morpho/Split
        - labels newlines
        - if splitSentences=false
    * annotation Morpho/SpaceToken
        - labels whitespaces within a sentence
    * annotation Morpho/Token
        - labels a word token
        - Features:
            * form
                - a word itself
            * lemma
                - a base form of the word (can contain a special dictionary suffix)
            * tag
                - a positional POS tag by Jan Hajiƒç's morphology dictionary tagset (http://ufal.mff.cuni.cz/pdt/Morphology_and_Tagging/Doc/hmptagqr.html)

Clean lemmas
-----------------
    Requirements:
    * JAPE Transducer

    This PR adds for each token a new "clean_lemma" feature, with a lemma stripped off its special dictionary suffix.

    Input:
    * annotation Morpho/Token
        - Features:
            * lemma

    Output:
    * annotation Morpho/Token
        - Features:
            * clean_lemma
                - the base form with its possible dictionary suffix stripped off
